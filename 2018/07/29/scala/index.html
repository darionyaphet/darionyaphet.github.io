Type Class

TypeClass 是 Haskell 中的概念，TypeClass 是定义公共行为的接口，类似于 Java 中的 Interface 和 Scala 中的 Trait。
TypeClass 将 行为定义 与 具有行为的对象 分离，更容易实现 DuckType; 在函数式编程中，通常将数据与行为相分离，甚至是数据与行为按需绑定，已达到更高级的组合特性。
Scala 使用隐式转换使之能够不那么完美地支持 TypeClass。

Monad

Monad 是一种将函子组合应用的方法，经常用来代表计算，用来把与业务无关的通用程序抽象出来。
